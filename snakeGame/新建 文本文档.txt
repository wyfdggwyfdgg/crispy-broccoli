项目介绍
这是一个使用Python的pygame库制作的简单贪吃蛇游戏。在游戏中，玩家需要控制一条蛇来捕获屏幕上出现的食物。每当捕获到食物后，蛇的长度会增加，同时得分增加。游戏设置了两种失败条件：蛇撞到屏幕边缘或者撞到自己的身体。当游戏失败时，系统会显示GameOver界面，玩家可以选择重新开始。

运行说明
确保你已经安装了Python环境。如果你没有安装Python，可以访问 Python官网 进行下载并安装。

通过终端或者命令提示符安装 pygame 库：

pip install pygame
将上面提供的代码保存为名为 snake_game.py 的文件。同时，请确保你的代码目录中包含了所需的以下资源文件：background.png（背景图片），snake_body.png（蛇身体图片），food.png（食物图片）。

在终端或命令提示符中，进入包含 snake_game.py 的文件夹，然后运行以下命令：

python snake_game.py
游戏现已开始。你可以使用键盘的上、下、左、右方向键来控制蛇的移动方向。当蛇捕获到食物时，蛇的长度会增加，得分也会相应提高。

当游戏结束时（蛇撞到屏幕边缘或者撞到自己的身体），GameOver界面将显示你的得分和操作提示。按 R 键重新开始游戏，按 Q 键或 Esc 键退出游戏。
